Mappamondo

Questo file SVG è una rappresentazione in proiezione ortografica della Terra. È stato costruito a partire dai dati di NaturalEarth, a cui sono state applicate delle trasformazioni attraverso il pacchetto Cartopy per Python. Nell'immagine ci sono in totale trentasei frame, uno ogni 10° di longitudine, che si alternano con una transizione CSS.

Questo SVG è stato creato in origine per un quiz sui vari Stati.

Di seguito una parte del codice della transizione:

Il frame numero 35:

.f35 {
animation: 36s linear 0s infinite anima;
}

L'animazione "anima", applicata a tutti i frame:

@keyframes anima {
0% {visibility: hidden}
97.22% {visibility: hidden}
97.23% {visibility: visible}
100% {visibility: visible}
}
